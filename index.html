Esse é o header do Article

Aqui é onde escrevo textos maiores, utilizando a tag P (paragraph) que representa um parágrafo, o ideal é sempre usar o P quando for escrever um texto grande pois as tags "H" são destinadas a títulos e subtitulos, e utilizar alguma delas para escrever parágrafos é anti-semântico e inviável para a acessibilidade, visto que leitores de tela utilizam as tags para se localizar e mapear o site para o usuário.
